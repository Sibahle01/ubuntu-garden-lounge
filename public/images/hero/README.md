# Hero Image Requirements

Please add your hero image as:
public/images/hero/main-background.jpg

Recommended specifications:
- Format: WebP or JPEG
- Size: 1920x1080px minimum
- Quality: High resolution, professional photography
- Content: Restaurant ambiance, interior, or garden setting
- Style: Warm lighting, inviting atmosphere

Alternative names you can use:
- main-background.jpg
- hero-image.jpg
- ambiance.jpg
- restaurant-interior.jpg

The image will be displayed as a full-screen background with an overlay for text readability.
